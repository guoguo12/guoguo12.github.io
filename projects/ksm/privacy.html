Keyboard Shortcuts for Messenger Privacy Policy

Keyboard Shortcuts for Messenger is a Chrome extension that adds shortcuts to Facebook's Messenger web app.

The extension does not collect, retain, or share user information in any way.

The extension does not interact with or transmit data to servers or networks belonging to its developers or affiliates.

The extension does not store or read cookies.

The source code of the extension is freely available on GitHub
