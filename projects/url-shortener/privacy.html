URL Shortener Privacy Policy

Introduction

This is the privacy policy page for the URL Shortener add-on for Google Docs, which is used to shorten long URLs using the Google URL Shortener service.

Policy

The URL Shortener accesses the following user data:
  • The URL or text selected at the time the user clicks "Shorten selected URL" in the URL Shortener menu.
  • The URL of the document, when the user clicks "Get document link" in the URL Shortener menu.
This user data is passed directly to the Google URL Shortener service, which (on success) replies with the requested short URL. The URL Shortener add-on does not communicate with any other external servers or databases.

When the URL of a document is shortened (using the "Get document link" feature), the resulting short URL is stored in the Properties object corresponding to the document. This object is not accessible by any other user or add-on.

You can find the privacy policy for the Google URL Shortener service at their website (click "Privacy Policy" in the footer).
